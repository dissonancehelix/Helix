# archive/

Raw provenance, legacy material, imports, analyses, and migration history.

Archive preserves evidence and project memory. It is not active canon.

## Contents

- `raw/` — raw source dumps and restore-only provenance.
- `legacy/` — old structure notes and deprecated material.
- `migrations/` — migration manifests and regeneration history.
- `imports/` — import review history.
- `analyses/` — archived analysis outputs.

Raw evidence is not deleted. Generated material does not become true by being
archived.
