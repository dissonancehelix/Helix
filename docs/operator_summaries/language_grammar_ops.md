# Language grammar drift (neutral cultural evolution)
State space: Population distribution of syntactic rules and lexical usage frequencies
Dynamics operator: Intergenerational transmission, stochastic learning, social interaction
Perturbation operator: Transmission errors, substrate interference, language contact, sociolinguistic innovation
Stability condition: Fixation or high-frequency equilibrium of grammatical variants
Failure mode: Grammatical restructuring, Creole formation, language death
Observables: Variant frequency, Rate of lexical replacement, Syntactic complexity
Timescale regime: Generations (decades to centuries)
Persistence type: PATTERN
Non-geometric elements: Arbitrariness of the sign, Discrete combinatorial syntax
Edge conditions: Population bottlenecks, Rapid substrate shift
Notes: Grammar persists through continuous imperfect replication without singular state attractors.
