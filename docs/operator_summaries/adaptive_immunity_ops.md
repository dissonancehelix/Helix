# Adaptive immune response (clonal selection)
State space: Repertoire of lymphocyte antigen receptors and antibody concentrations
Dynamics operator: Antigen presentation, clonal expansion, somatic hypermutation, affinity maturation
Perturbation operator: Pathogen invasion, mutational escape, self-antigen exposure
Stability condition: Pathogen clearance rate exceeds pathogen replication rate
Failure mode: Immune exhaustion, immune evasion, or autoimmune cascade
Observables: Antibody titers, T-cell clonal frequencies, Cytokine levels, Pathogen load
Timescale regime: Days to weeks (clonal expansion), years (memory persistence)
Persistence type: MIXED
Non-geometric elements: V(D)J recombination combinatorics, Sequence space search
Edge conditions: Immunosenescence, High pathogen mutation rates (e.g., HIV)
Notes: Maintenance and perturbation involve coupled sequence-space evolution.
