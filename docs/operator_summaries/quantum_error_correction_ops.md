# Quantum Error Correction Codes (surface code)
State space: Hilbert space with logical qubit subspace defined by stabilizer syndromes
Dynamics operator: Unitary evolution and projective syndrome measurements
Perturbation operator: Pauli errors (X, Y, Z)
Stability condition: Error weight < code distance d/2
Failure mode: Logical Pauli operator formation (unrecoverable state change)
Observables: Syndrome measurements, Logical error rate, Code distance d
Timescale regime: Discrete syndrome measurement cycles
Persistence type: STATE
Non-geometric elements: Topological code distance, Projective measurement collapse
Edge conditions: Error probability approaching physical threshold, Measurement latency
Notes: Topological invariant protects information; stability scales with code distance.
