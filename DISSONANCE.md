COGNITIVE COORDINATE SYSTEM
------------------------------------------------------------


The operator encodes and navigates systems through a 6-axis basis:

1. Attractor Stability
   Degree to which structural patterns recur identically across the entity
   0 = every phrase novel | 1 = exact loop

2. Generative Constraint
   Degree to which the pitch/rhythm/timbre space is restricted by compositional rules
   0 = unconstrained (free/chromatic) | 1 = single scale, strict grid

3. Recurrence Depth
   Hierarchical depth of detectable recurrence (motif → phrase → section → form)
   0 = no recurrence | 1 = all four levels confirmed

4. Structural Density
   Rate of structural events per unit time (note density, harmonic rhythm, timbre change)
   0 = sparse, static | 1 = maximum density for domain

5. Control Entropy
   Entropy of control signal distributions (pitch, dynamics, articulation)
   0 = uniform/constant | 1 = maximally varied distribution

6. Basin Permeability
   Ease of transition between structural states
   0 = hard cuts only | 1 = fully continuous, no distinct sections
