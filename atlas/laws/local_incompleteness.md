# Law: Local Incompleteness

**Confidence:** Exploratory
**Last Updated:** 2026-03-15
**Probe Version:** not yet implemented

---

## Description

Complex adaptive systems contain regions where local information is
structurally insufficient to determine global behavior. No finite local
observation window can predict certain global transitions. This is not
a measurement problem — it is a structural property of the system's
topology.

Candidate instances: cellular automata boundary effects, network
partition effects, linguistic ambiguity zones.
