# Law: Epistemic Irreversibility

**Confidence:** Exploratory
**Last Updated:** 2026-03-15
**Probe Version:** not yet implemented

---

## Description

Once a system has incorporated a piece of information that resolves
a prior ambiguity, it cannot return to a state of genuine unknowing
about that distinction. The epistemic state space is asymmetrically
structured — learning is not reversible in the same way physical
processes can be reversed.

This pattern appears in cognitive models, language acquisition, and
decision systems. It may be a structural consequence of how information
reduces entropy in state spaces.
