# model/

Modeled understanding lives here.

- `map/` is the machine-readable cognitive map: patterns, gates, examples, probes, anomalies, links, and source registry.
- `domains/` is modeled domain understanding: music, games, language, food, software, and other interpretation chambers.

`model/` does not store raw evidence and does not host tools. It compresses what has already survived observation, domain modeling, and review.
