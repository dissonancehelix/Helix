# Trails Domain

## Domain Role

Trails/Kiseki is a first-class externalized cognition domain because it has become database, wiki, continuity map, character/institution memory, and ingestion project. It is one of the strongest examples of world-memory made operational.

## External / Operational Model

Trails is modeled as:

- a continuity engine across games, regions, institutions, factions, and arcs
- a character-memory system where observer evolution and hidden-structure reveals matter
- a database/wiki/export project with source ingestion and reusable structure
- a world where institutions, hidden organizations, and long payoff chains preserve state across time

Do not duplicate all general games logic here. Field mechanics and play-fit rules live in `model/domains/games/`; Trails focuses on world-memory and continuity architecture.
