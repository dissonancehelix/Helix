# Domain: Language

This is a Structured Knowledge Surface. It represents a formal, modeled reality of linguistics, syntax, and structural language representation.

**Constraints:**
- This is not an application.
- This is not a research lab.
- This directory contains normalized models, Wikipedia semantic schemas, and grammar rules.
- Any tool built to scrape Wikipedia or process text belongs in `system/tools/`.
- Any experiment testing linguistic constraints belongs in `labs/`.
