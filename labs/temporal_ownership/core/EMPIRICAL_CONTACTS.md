# Empirical Contacts

This file lists research areas that can pressure-test ICT. It does not claim any area proves ICT.

## No-Report Paradigms

- Why relevant: Separates consciousness from explicit report.
- Tests primitives: field, reportability, appearance-from-within.
- Would support ICT: Evidence of conscious presence without report strengthens report-insufficiency.
- Would weaken ICT: If no-report markers collapse into report preparation only.

## Anesthesia Transitions

- Why relevant: Shows changes in field, access, integration, and re-entry.
- Tests primitives: field, integration, continuity, appearance-from-within.
- Would support ICT: Graded loss/recovery patterns that separate integration, access, and self-continuity.
- Would weaken ICT: If reportability alone tracks all relevant transitions.

## Sleep / Dreaming

- Why relevant: Tests continuity through altered or absent report.
- Tests primitives: field, continuity, self-model, reportability.
- Would support ICT: Dream or waking re-entry patterns where report is delayed but field/self markers persist.
- Would weaken ICT: If sleep continuity is fully explained by later narrative reconstruction.

## Disorders Of Consciousness

- Why relevant: Separates wakefulness, awareness, report, and covert cognition.
- Tests primitives: field, reportability, integration, agency/control coupling.
- Would support ICT: Covert consciousness cases where report is absent or minimal but field-like markers remain.
- Would weaken ICT: If all apparent covert consciousness is reducible to non-conscious processing.

## Locked-In / Covert Consciousness

- Why relevant: Strong report limitation with potentially preserved interiority.
- Tests primitives: reportability, agency/control coupling, field, ownership.
- Would support ICT: Evidence of owned experience despite blocked outward action.
- Would weaken ICT: If report/action channels are necessary for all ownership evidence.

## Body Ownership Illusions

- Why relevant: Manipulates ownership and body boundary.
- Tests primitives: ownership, embodiment, boundary, self-model.
- Would support ICT: Ownership can shift without full identity shift, showing ownership as distinct primitive.
- Would weaken ICT: If ownership reports have no independent behavioral or neural structure.

## Agency Illusions

- Why relevant: Separates action, authorship, and report.
- Tests primitives: agency/control coupling, ownership, self-model.
- Would support ICT: Cases where agency can be misassigned while field remains, sharpening ownership.
- Would weaken ICT: If agency attribution is only post-hoc report with no primitive role.

## Depersonalization / Derealization

- Why relevant: Directly pressures ownership, reality-feel, and self-world boundary.
- Tests primitives: interior, ownership, boundary, appearance-from-within.
- Would support ICT: Preserved perception with weakened ownership/interiority distinction.
- Would weaken ICT: If these states reduce entirely to verbal framing.

## Split-Brain

- Why relevant: Tests whether field and ownership can divide.
- Tests primitives: integration, boundary, ownership, branching.
- Would support ICT: Evidence for partial interiors or fractured ownership.
- Would weaken ICT: If strong disconnection has no effect on ownership/field structure.

## Dissociation

- Why relevant: Tests state-dependent ownership and continuity.
- Tests primitives: self-model, ownership, continuity, boundary.
- Would support ICT: Distinct continuity profiles within one organism.
- Would weaken ICT: If dissociative differences reduce entirely to memory/report artifacts.

## Amnesia / Dementia

- Why relevant: Separates memory, continuity, self-model, and field.
- Tests primitives: memory continuity, ownership, self-model, continuity.
- Would support ICT: Persistence of owned field despite memory damage, plus graded self-continuity loss.
- Would weaken ICT: If memory similarity fully explains identity and ownership.

## Infants / Animals

- Why relevant: Tests consciousness and selfhood without adult report.
- Tests primitives: field, embodiment, agency/control coupling, reportability.
- Would support ICT: Nonverbal markers of owned perspective or boundary maintenance.
- Would weaken ICT: If report-like capacities are required for all credible consciousness attribution.

## Perturbational Complexity / PCI

- Why relevant: Measures complexity/integration during conscious and unconscious states.
- Tests primitives: integration, field, appearance-from-within.
- Would support ICT: Integration/complexity tracks field presence better than report alone.
- Would weaken ICT: If PCI-like measures do not distinguish relevant states.

## Neural Integration Measures

- Why relevant: Tests whether unity requires integrated dynamics.
- Tests primitives: integration, field, boundary.
- Would support ICT: Integrated dynamics predict field unity or loss.
- Would weaken ICT: If integration has no relation to field or ownership.

## Recurrent Processing Evidence

- Why relevant: Tests local perceptual presence.
- Tests primitives: field, perceptual presence, reportability.
- Would support ICT: Perceptual presence can occur without full global broadcast.
- Would weaken ICT: If recurrence is neither necessary nor informative for presence.

## Global Workspace Evidence

- Why relevant: Tests access, report, and flexible control.
- Tests primitives: reportability, access, agency/control coupling.
- Would support ICT: Broadcast explains report while leaving ownership/A as separate targets.
- Would weaken ICT: If global availability fully explains field, ownership, and continuity.

## Predictive Processing / Active Inference Models

- Why relevant: Tests boundary maintenance, agency, and self-world modeling.
- Tests primitives: boundary, self-model, agency/control coupling, embodiment.
- Would support ICT: Boundary/control models predict ownership and self-disturbance.
- Would weaken ICT: If predictive/control models fully solve appearance without residue.

## LLMs / AI Agents As Report-Without-Ownership Stress Tests

- Why relevant: Systems can report selfhood without clear ownership or continuity.
- Tests primitives: reportability, self-model, ownership, appearance-from-within.
- Would support ICT: Persistent reports remain insufficient unless ownership/continuity tests are met.
- Would weaken ICT: If self-report plus tool agency becomes sufficient evidence without further primitives.
