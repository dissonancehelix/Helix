# Falsifiers And Hard Weakening Cases

ICT should weaken when another explanation absorbs its primitives without residue. A falsifier does not need to insult the theory; it needs to make the theory unnecessary, overfit, or wrongly separated.

## Hard Weakening Cases

1. Reportability fully explains consciousness and selfhood.
   - Weakens: appearance-from-within, ownership, continuity as separate primitives.
   - Update: Demote ICT to a vocabulary for report structure.

2. Ownership can be reduced entirely to report.
   - Weakens: ownership as primitive.
   - Update: Replace ownership tests with report/access tests.

3. Temporal continuity is unnecessary for selfhood.
   - Weakens: owned continuity as the self criterion.
   - Update: Treat selfhood as momentary subject-model rather than persistence.

4. Branching does not affect first-person identity in any meaningful way.
   - Weakens: branching threat and singular continuation claims.
   - Update: Reframe identity as branch-relative pattern continuation.

5. Self/world boundary is unnecessary.
   - Weakens: boundary and interior primitives.
   - Update: Remove boundary maintenance as a core test.

6. Integration alone explains consciousness, ownership, and selfhood with no residue.
   - Weakens: ICT's separation between field unity and owned continuity.
   - Update: Collapse ICT into an integration theory with continuity annotations.

7. Self-modeling is unnecessary for all selfhood.
   - Weakens: self-model as a major primitive.
   - Update: Treat self-model as optional reflective layer, not selfhood support.

8. Memory similarity fully explains identity persistence.
   - Weakens: continuity, ownership, and transformation distinctions.
   - Update: Replace continuity tests with memory similarity tests.

9. Current or future AI self-report becomes sufficient evidence for inhabited selfhood without continuity or ownership tests.
   - Weakens: report-without-ownership stress test.
   - Update: Demote ownership and continuity gates for artificial systems.

10. Appearance-from-within becomes fully explained without field, boundary, or ownership concepts.
    - Weakens: A as central gap and ICT's primitive set.
    - Update: Preserve any useful operational tests, but demote the theory's core framing.

## Demotion Rule

- If a claim survives only as metaphor, mark it `analogy_only`.
- If a claim cannot name a weakening condition, mark it `speculative`.
- If a claim repeatedly overpredicts ownership from report, add a negative control.
- If a claim works only in biological cases, restrict it instead of generalizing it.
- If a claim works only in artificial cases, restrict it instead of back-projecting it onto human selfhood.

## Strong Revision Triggers

- Clear dissociation between memory and selfhood in either direction.
- Credible evidence of conscious appearance without report, memory, or self-reference.
- Credible evidence of robust report, memory, and self-reference without ownership.
- Branching cases that force identity to become plural, probabilistic, or non-singular.
- Integration measures that predict appearance but not ownership.
