# Inhabited Interiority Research

This folder holds the small-first research machinery for the Temporal Ownership Model.

It is not canon, not the theory itself, and not a proof machine. It connects conceptual tests to datasets, literature, analysis lanes, falsifiers, and small pilots.

Prompts generate hypotheses; datasets apply pressure; ledgers prevent delusion.

## Temporal Ownership Model

Temporal Ownership Model is the umbrella / integration model for the current research program.

Best compression:

> Temporal Ownership asks how experience becomes someone's across time.

Second compression:

> LIP is before the crossing, DCP is the crossing, EIP is the residue after the crossing, and ICT asks whether the crossing was owned.

Working hierarchy:

```text
Temporal Ownership Model
├── ICT: ownership, selfhood, continuity, appearance-from-within
├── DCP: compression, commitment, residue
│   ├── LIP: local incompleteness before commitment
│   └── EIP: epistemic irreversibility after commitment
└── External comparison theories
    ├── IIT
    ├── GNWT/GWT
    ├── RPT
    ├── HOT
    ├── Predictive Processing / Active Inference
    └── Self-Model Theory
```

## Roles

- ICT = theory / hypothesis family about owned continuity, selfhood, and appearance-from-within.
- DCP = principle / transition model about compression, decision boundaries, irreversible commitment, and residue.
- LIP = Local Incompleteness Principle: before commitment, local observation is insufficient under hidden state, partial observability, rotating threats, or nonlocal consequences.
- EIP = Epistemic Irreversibility Principle: after commitment, observation, decision, learning, or transformation leaves irreversible epistemic residue; the prior state cannot be fully restored.

Core relation:

- LIP is before commitment.
- DCP is at commitment.
- EIP is after commitment.
- ICT is the ownership layer: did anyone live/own that transition from within?

## External Theory Integration

- IIT: binding / intrinsic integration / field candidate.
- GNWT/GWT: access / broadcast / reportability.
- RPT: recurrent perceptual presence.
- HOT: reflective awareness / meta-representation.
- Predictive Processing / Active Inference: boundary, prediction, and control coupling.
- Self-Model Theory: subject structure, ownership model, and perspective.
- DCP/LIP/EIP: transition, incompleteness, commitment, irreversibility, and residue.
- ICT: owned continuity through time.

## Caution

DCP may partly formalize the operator's own compression-native cognition. That makes it productive but risky. It must be checked against external data, negative controls, and falsifiers so it does not become self-confirming recursion.

If DCP resembles the operator's cognition, that is signal, not proof.

No report-level claim is promoted automatically.

## Small-First Strategy

Start with papers, documentation, metadata, and one tiny pilot. Do not download huge raw datasets by default.

Priority:

1. DREAM metadata / smallest labeled subset.
2. OpenNeuro ds004902 metadata and one-subject EEG pilot only.
3. COGITATE design and variable map only.
4. OpenNeuro ds006623 metadata, participant info, propofol profiles, and squeeze traces before imaging.
