# Appearance–Ownership–Continuity Framework

Appearance–Ownership–Continuity Framework (AOC Framework) is the active cross-domain research room for appearance-from-within, ownership, commitment, access, binding, report, selfhood, and continuity.

## Naming

Current umbrella:

> Appearance–Ownership–Continuity Framework (AOC Framework)

Deprecated working titles:

- Temporal Ownership Model / TOM
- Inhabited Continuity Theory / ICT

Current sub-hypothesis:

> Owned Continuity Hypothesis (OCH)

## Active Model

AOC Framework is the umbrella / integration model.

- `THEORY.md` is the root orientation.
- `core/` is the active conceptual harness for OCH/AOC tests.
- `research/` contains datasets, literature, small-first data triage, and analysis lanes.
- Reports are evidence, not canon.
- Claims are promoted only through review.
- Obsolete or superseded docs are deleted after useful content is merged.

Working compression:

> LIP is before the crossing, DCP is the crossing, EIP is the residue after the crossing, and OCH asks whether the crossing was owned.

## Current Hierarchy

```text
Appearance–Ownership–Continuity Framework
├── OCH / Owned Continuity Hypothesis
│   └── owned continuity, selfhood, appearance-from-within
├── DCP / Decision-Compression Principle
│   └── compression, decision boundary, commitment, residue
│       ├── LIP / Local Incompleteness Principle
│       │   └── pre-commitment local insufficiency
│       └── EIP / Epistemic Irreversibility Principle
│           └── post-commitment epistemic residue
└── External comparison theories
    ├── IIT
    ├── GNWT/GWT
    ├── RPT
    ├── HOT
    ├── Predictive Processing / Active Inference
    └── Self-Model Theory
```

External theories are comparison pressure, not replacements:

- IIT: binding / intrinsic integration / field candidate.
- GNWT/GWT: access / broadcast / reportability.
- RPT: recurrent perceptual presence.
- HOT: reflective awareness / meta-representation.
- Predictive Processing / Active Inference: boundary, prediction, control coupling.
- Self-Model Theory: subject structure, ownership model, perspective.
