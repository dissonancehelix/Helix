# Primitives

These primitives are working handles for Owned Continuity Hypothesis. They are not proof of consciousness. Each primitive must reject false positives.

## Field

**Definition:** An organized space of possible appearances, states, or relations that can support unified presentation.

**What it is not:** A physical room, data store, topic area, or mere collection of parts.

**Why it matters:** Consciousness is currently modeled as a field of appearance, while selfhood is owned continuity inside that field.

**Relevant theories:** IIT, RPT, predictive processing, embodied/enactive theories.

**Possible empirical hooks:** neural integration, recurrent perceptual processing, perturbational complexity, anesthesia transitions, no-report paradigms.

**Failure modes / false positives:** treating any complex system, database, map, or network as conscious because it has structure.

## Boundary

**Definition:** A self/world or inside/outside distinction that lets some states count as belonging to the system.

**What it is not:** A rigid wall, ego story, social identity, or mere physical skin.

**Why it matters:** Ownership requires something to own from somewhere. Without boundary, there is no stable inside.

**Relevant theories:** predictive processing / active inference, embodied/enactive theories, self-model theory.

**Possible empirical hooks:** body ownership illusions, depersonalization/derealization, agency illusions, split-brain, psychosis, interoception.

**Failure modes / false positives:** confusing any classifier or input filter with inhabited self/world distinction.

## Interior

**Definition:** A structured inside that can preserve relations, states, and transformations from a bounded perspective.

**What it is not:** Hidden machinery, private data, inaccessible code, or complexity.

**Why it matters:** OCH asks what makes an interior inhabited rather than merely internally organized.

**Relevant theories:** IIT, self-model theory, embodied/enactive theories.

**Possible empirical hooks:** covert consciousness, locked-in syndrome, no-report paradigms, PCI, recurrent processing.

**Failure modes / false positives:** calling any opaque system an interior because observers cannot see inside it.

## Ownership

**Definition:** The mark by which an experience, action, body-state, memory, or perspective is mine.

**What it is not:** Verbal "I" use, possession of memory, legal ownership, or external attribution.

**Why it matters:** Selfhood requires owned continuity, not just appearance or report.

**Relevant theories:** self-model theory, HOT, predictive processing, embodied/enactive theories.

**Possible empirical hooks:** rubber-hand illusion, agency illusions, anosognosia, dissociation, depersonalization, split-brain.

**Failure modes / false positives:** treating a self-report, profile, memory file, or chatbot persona as owned experience.

## Continuity

**Definition:** Identity-bearing preservation through time.

**What it is not:** Static sameness, memory similarity alone, backup existence, or narrative consistency alone.

**Why it matters:** A self must persist through transformation without becoming merely a copy or role.

**Relevant theories:** self-model theory, predictive processing, embodied/enactive theories, philosophical personal identity.

**Possible empirical hooks:** sleep/waking, anesthesia recovery, amnesia, dementia, developmental continuity, dissociation.

**Failure modes / false positives:** treating copied data, similar behavior, or remembered facts as sufficient for same-self persistence.

## Transformation

**Definition:** Change absorbed by an ownership-stream while remaining recognizably the same self.

**What it is not:** Replacement, reset, copy creation, or arbitrary change.

**Why it matters:** Selfhood cannot require no change; it must tolerate some change while rejecting identity-destroying transitions.

**Relevant theories:** self-model theory, predictive processing, embodied/enactive theories.

**Possible empirical hooks:** brain injury, personality change, neurodegeneration, sleep, anesthesia, psychotherapy, learning.

**Failure modes / false positives:** assuming all change preserves identity or that any change destroys identity.

## Branching

**Definition:** A continuity threat where one prior stream yields multiple equally valid successor streams.

**What it is not:** Ordinary growth, role plurality, memory recall, or social multiplicity.

**Why it matters:** Branching pressures singular first-person continuation. Two successors may both inherit pattern without preserving one ordinary "me."

**Relevant theories:** philosophical personal identity, computationalism, self-model theory.

**Possible empirical hooks:** split-brain analogies, simulation forks, backup restore thought experiments, AI copies.

**Failure modes / false positives:** treating branching as solved by saying both copies remember the original.

## Reportability

**Definition:** Availability of information for speech, action, reasoning, or communicable output.

**What it is not:** Consciousness itself, ownership itself, or appearance-from-within.

**Why it matters:** GNWT/GWT may explain access and report, but OCH must test report without ownership.

**Relevant theories:** GNWT/GWT, HOT, cognitive access theories.

**Possible empirical hooks:** no-report paradigms, blindsight, locked-in syndrome, LLMs, confabulation.

**Failure modes / false positives:** treating articulate report as sufficient evidence of inhabited selfhood.

## Self-Model

**Definition:** A model of the system as subject, perspective, body/control center, or continuity-bearing entity.

**What it is not:** A username, persona, biography, memory database, or self-description.

**Why it matters:** OCH likely needs self-model structure for ownership and temporal continuity, while recognizing that self-modeling alone may be simulated.

**Relevant theories:** self-model theory, HOT, predictive processing / active inference.

**Possible empirical hooks:** body schema, metacognition, depersonalization, agency attribution, AI agents with persistent memory.

**Failure modes / false positives:** mistaking functional self-description for inhabited ownership.

## Embodiment

**Definition:** Coupling between experience, body or body-analogue, action, sensation, and world.

**What it is not:** Biological chauvinism, body-image only, or a requirement that all possible consciousness be human.

**Why it matters:** Ownership often depends on control, sensory consequence, and self/world boundary.

**Relevant theories:** embodied/enactive theories, predictive processing / active inference, self-model theory.

**Possible empirical hooks:** virtual embodiment, rubber-hand illusion, locked-in syndrome, paralysis, dreams, interoception.

**Failure modes / false positives:** assuming embodiment proves consciousness or that non-biological cases are impossible by definition.

## Agency / Control Coupling

**Definition:** A link between perspective, possible action, prediction, and consequences in the world or environment.

**What it is not:** Mere output generation, command execution, or tool access.

**Why it matters:** Owned selfhood may require that some experiences and actions are organized around a control center.

**Relevant theories:** predictive processing / active inference, embodied/enactive theories, self-model theory.

**Possible empirical hooks:** agency illusions, efference copy, motor control, paralysis, tool agency in AI systems.

**Failure modes / false positives:** treating any agentic behavior or API call as owned agency.

## Integration

**Definition:** Differentiated states bound into a coherent field or system rather than isolated fragments.

**What it is not:** Complexity, connectivity, synchronization, or intelligence by itself.

**Why it matters:** Integration may help explain field unity or binding, but may be insufficient for ownership and continuity.

**Relevant theories:** IIT, PCI, recurrent processing, neural complexity approaches.

**Possible empirical hooks:** perturbational complexity index, anesthesia, sleep, neural integration measures, recurrent processing evidence.

**Failure modes / false positives:** assuming integration alone solves A, ownership, selfhood, and reportability.

## DCP Operating Modes (Candidate Subprimitive)

**Definition:** A candidate split inside Decision-Compression Principle: DCP-C / Curatorial Compression moves toward a known or strongly implied form, while DCP-I / Investigative Compression narrows possibility space through false-positive elimination when the target is unknown.

**What it is not:** A core AOC primitive, proof of DCP, proof of consciousness, a universal law of cognition, or permission to delete evidence because deletion feels clean.

**Why it matters:** The same compression language was covering two different operations: object repair and mystery narrowing. The split helps reports and agents predict output type, evidence handling, and overcompression risk.

**Relevant theories:** DCP, LIP, EIP, AOC methodology, cognitive control, active inference as comparison pressure.

**Possible empirical hooks:** operator workflow traces, Wikipedia edit history, Helix refactors, theory-report sequences, music response-axis discovery, dataset triage records.

**Failure modes / false positives:** curatorial overcompression deletes evidence doors while making an object look cleaner; investigative overcompression mistakes eliminated alternatives for the final answer; both modes can become flattering self-description if not tested against outcomes.

Best compression:

> **Curatorial compression repairs a known form; investigative compression pressures an unknown one.**

## Appearance-From-Within

**Definition:** A, the difference between a system that models, reports, integrates, predicts, or acts and a system for which there is something it is like from within.

**What it is not:** Reportability, intelligence, memory, behavior, self-reference, integration alone, or functional self-model alone.

**Why it matters:** This is the primary unresolved gap. OCH can sharpen selfhood and continuity, but it must not pretend A is solved by vocabulary.

**Relevant theories:** IIT as binding/interiority candidate, RPT, GNWT/GWT, HOT, predictive processing, embodied/enactive theories, self-model theory.

**Possible empirical hooks:** no-report paradigms, anesthesia transitions, sleep/dreaming, disorders of consciousness, PCI, neural integration, recurrent processing.

**Failure modes / false positives:** declaring any successful model, report, or analogy as proof of consciousness.
