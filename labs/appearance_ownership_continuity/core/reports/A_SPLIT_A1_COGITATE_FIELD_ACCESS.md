## 2. Definitions From PRIMITIVES.md

- **A1 fielded appearance:** appearance organized as a unified or structured field rather than disconnected fragments.
- **Field:** an organized space of possible appearances, states, or relations that can support unified presentation.
- **Integration:** differentiated states bound into a coherent field or system rather than isolated fragments.
- **Reportability:** availability of information for speech, action, reasoning, or communicable output.
- **Access:** operational availability for reasoning, task control, report, or flexible behavior.
- **Appearance-from-within:** the difference between a system that models, reports, integrates, predicts, or acts and a system for which there is something it is like from within.
- **Ownership:** the mark by which an experience, action, body-state, memory, or perspective is mine.
- **Task relevance:** whether a stimulus feature matters for the task, report, decision, or behavioral response.
- **Residue / marker:** an observable trace after or during a process; useful evidence, but not the target itself.

## 3. COGITATE Design Map

COGITATE is an open-science, preregistered adversarial collaboration comparing IIT and GNWT using fMRI, MEG/M-EEG, and iEEG across multiple research institutions. The 2025 Nature paper reports a direct IIT/GNWT comparison with 256 human participants viewing suprathreshold visual stimuli while neural activity was measured. Reported anchors include conscious-content information in visual, ventrotemporal, and inferior frontal cortex; sustained responses in occipital/lateral temporal cortex reflecting stimulus duration; and content-specific synchronization between frontal and early visual areas.

| Variable / channel | AOC classification | Use | Main caution |
| --- | --- | --- | --- |
| stimulus/content category | A1 field/content pressure | content-structure contrast | content classification is not appearance |
| stimulus duration | A1 content persistence pressure | duration-linked maintenance | persistence marker is not field unity by itself |
| task relevance | access/report pressure | separates task demand from content | task relevance can masquerade as consciousness |
| orientation | content-feature and task pressure | separates perceptual feature from task handling | feature coding can become task coding |
| fMRI | marker/residue pressure | spatial content/access localization | BOLD region is not appearance |
| MEG/M-EEG | marker/residue pressure | temporal dynamics and synchronization | timing marker is not lived presence |
| iEEG/ECoG | marker/residue pressure | local content and connectivity detail | intracranial detail is still a marker |
| report/behavioral response | access/report pressure | output and task residue | report is not appearance |
| neural content information | A1 content marker pressure | content-specific activity | decoded content is not experience itself |
| sustained activity/duration response | A1 persistence pressure | best field-persistence neighbor | sustained activity may be task/access residue |
| synchronization/connectivity | field/access ambiguity pressure | tests inter-area coordination | connectivity can reflect access route, task demand, or content marker |

### Prediction Family Map

- **IIT:** pressures integrated/intrinsic content structure and sustained field candidates.
- **GNWT/GWT:** pressures access, broadcast, task relevance, and report-linked availability.
- **RPT-neighbor:** pressures recurrent perceptual presence and duration/content persistence.
- **AOC interpretation:** keeps A1 field/content pressure distinct from A0 raw appearance and A2 ownership.

## 5. Report vs No-Report Guardrail

No-report paradigms matter because report requirements can bias neural-correlate work toward the neural correlates of perceptual report rather than phenomenal experience. They are useful subtraction tools: by reducing explicit report demands, they can help separate access/report machinery from content markers.

But no-report paradigms are not magic. They still rely on proxy markers, calibration assumptions, task design, eye movements, autonomic markers, or indirect behavior. AOC should use no-report logic as a guardrail, not as an escape hatch. The correct lesson is not "remove report and the remainder is A"; it is "report is one channel among several, and each channel can contaminate interpretation."

For COGITATE, this means task relevance, behavior, and report should be read as access/residue channels, while content, duration, and synchronization should be read as field/access ambiguity channels. None of them is raw appearance by itself.

## 6. A1 Pressure

COGITATE can pressure:

- content persistence,
- field/content representation,
- duration-linked sustained activity,
- access/report versus field distinction,
- theory comparison between IIT and GNWT,
- ambiguity between content marker and experience marker.

COGITATE cannot pressure directly:

- A0 raw appearance,
- A2 ownership/mineness,
- A3 lived temporal flow,
- A4 re-entry,
- A5 identity preservation,
- inhabited selfhood.

The best A1 use is disciplined separation. COGITATE gives several channels that look close to A1: content information, sustained duration-linked activity, and synchronization. It also gives several channels that can mimic A1: task relevance, report, behavioral response, and access-linked frontal involvement. The report becomes useful for AOC precisely because it does not let those channels collapse.

The three pilots triangulate different proxy failures:

- Propofol blocks the response shortcut.
- DREAM blocks the retrospective-report shortcut.
- COGITATE blocks the content/access-marker shortcut.

Together, they make A-Split more useful: A1 field/content, A4 re-entry, reportability, responsiveness, and marker residue cannot be treated as one explanatory object.

## 9. Escalation Decision

Full COGITATE raw download is not warranted now. The design-level pressure is already useful, and AOC has not yet named a marker-level question that requires raw fMRI, MEG/M-EEG, or iEEG/ECoG.

- Do posterior/temporal markers better preserve content structure independent of report?
- Can COGITATE variables be re-coded into AOC channels without raw data first?

Near-term escalation, if needed later: small docs-only expansion or one-modality planning note. No raw payload yet.

## 10. Main Finding

COGITATE is useful because it pressures the field/access boundary without resolving raw A. It shows that IIT/GNWT-style theory predictions can be mapped onto separable AOC channels, but neural content markers, sustained activity, synchronization, report, and access remain markers rather than appearance itself.

The strongest conservative result is channel separation: COGITATE helps AOC ask whether content persistence and access/report are moving together or apart, without turning either into ownership or raw appearance.
