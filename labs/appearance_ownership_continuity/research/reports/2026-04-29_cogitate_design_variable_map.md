# COGITATE Design Variable Map

## Scope

Docs/design-variable map only. No raw fMRI, M-EEG, iEEG/ECoG, behavior, eye-tracking, or large BIDS payloads were downloaded.

Sources recorded locally in `labs/appearance_ownership_continuity/research/data_index/cogitate_docs/LINKS.md`.

## Design Variables

COGITATE is useful here because its design separates content, task relevance, duration, orientation, report/behavior, modality, and eye-tracking context.

| Variable | Levels / shape | AOC use |
| --- | --- | --- |
| Modality | fMRI, M-EEG, iEEG/ECoG, behavior, eye tracking | separates spatial, temporal, intracranial, behavioral, and gaze evidence channels |
| Stimulus category | face, object, letter, false font | content identity / category structure |
| Stimulus duration | 500 ms, 1000 ms, 1500 ms | content persistence and temporal maintenance |
| Orientation | center/front, left, right | perceptual feature / task-coded orientation |
| Task relevance | task-relevant target, task-relevant non-target, task-irrelevant | access/report/task pressure split |
| Behavioral response | button press / detection | reportability and task performance residue |
| Eye tracking | fixation, gaze, pupil where available | attention/control context, not ownership proof |

## Theory Pressure Map

| Theory / model | What COGITATE can pressure |
| --- | --- |
| GNWT / GWT | access, reportability, task relevance, behavioral availability, possible frontal/broadcast signatures |
| IIT | sustained posterior/temporal content structure, integration, intrinsic-structure candidates |
| RPT | recurrent perceptual presence, duration/content persistence before full report |
| AOC / OCH | not a direct ownership test; useful for separating access/report/binding/content duration from appearance-from-within |
| DCP | stimulus/report/task boundary as a commitment/residue surface |
| LIP | local neural signatures are insufficient without task/report/context |
| EIP | behavioral report and data record are irreversible residue after the trial |

## What COGITATE Can Pressure

- whether task relevance and reportability can be separated from content maintenance,
- whether duration-sensitive content signatures behave differently from report/task signatures,
- whether IIT/GNWT/RPT predictions divide across variables in ways useful to AOC,
- whether content, access, and duration are neighbors of appearance-from-within rather than identical to it.

## What COGITATE Cannot Pressure

- owned self-continuity,
- appearance-from-within directly,
- whether a conscious field is owned across time,
- whether DCP, LIP, or EIP are true,
- whether report/task success is sufficient for consciousness.

## Strongest AOC-Relevant Variables

1. Task relevance: best access/report pressure variable.
2. Stimulus duration: best content-maintenance / temporal-persistence variable.
3. Category and identity: best content-structure variable.
4. Behavioral response: best residue/report variable.
5. Modality: best comparison-theory pressure surface.

## Smallest Future Raw Subset

Defer raw data. If approved later, the smallest useful subset would be one modality plus one subject/run with behavioral events and eye-tracking context, chosen only after a specific question is written.

## No Claim Promotion

COGITATE is a theory-comparison benchmark, not a AOC proof source.
