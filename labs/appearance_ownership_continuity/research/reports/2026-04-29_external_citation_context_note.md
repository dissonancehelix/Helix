# External Citation Context Note

**Status:** source-backed context, not canon  
**Mode:** external pressure gathering  
**Scope:** AOC/A-Split support material for upcoming pressure reports  
**No claims promoted:** yes  

This note records where the Life Science Research plugin is useful for AOC work: it can supply outside anchors, dataset source lineage, and literature-pressure context. It does not replace the A-Split reports, claim ledger, or core theory files.

## Useful Role

The plugin is useful as a sensor for:

- public dataset and paper verification,
- PubMed/PMC/Nature/OpenNeuro-style citation anchors,
- review/literature pressure around body ownership, no-report paradigms, anesthesia, dreams, and temporal memory,
- future dataset-discovery passes when a pressure report needs an external anchor.

It should not drive:

- AOC/OCH claim promotion,
- A-Split layer definitions,
- DCP/LIP/EIP structure,
- local Helix ontology decisions.
