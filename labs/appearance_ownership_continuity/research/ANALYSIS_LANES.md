# Analysis Lanes

These lanes keep the Appearance–Ownership–Continuity Framework connected to external pressure without treating any one dataset as proof.

## 1. Conscious Content / Access / Binding

Core question:

> What distinguishes conscious content, access/report, and possible binding/interiority?

Candidate datasets:

- COGITATE

Relevant theories:

- IIT
- GNWT/GWT
- RPT
- OCH

Possible proxies:

- content decoding,
- duration response,
- task relevance,
- frontal/posterior signals.

False positives:

- content decoding equals ownership.

## 2. Report Vs Experience

Core question:

> When does experience plausibly occur without immediate report, and when does report fail to prove ownership?

Candidate datasets:

- DREAM
- anesthesia ds006623
- LLM stress tests

False positives:

- fluent report equals consciousness.

False negatives:

- missing report equals absence.

## 3. Interruption And Re-Entry

Core question:

> What returns after sleep, anesthesia, fainting, or altered states?

Candidate datasets:

- DREAM
- anesthesia ds006623

Possible proxies:

- connectivity recovery,
- imagery task response,
- report delay,
- re-entry markers.

False positives:

- memory return equals same owner.

## 4. Temporal Modeling Vs Temporal Experience

Core question:

> What separates representing time from undergoing time?

Candidate datasets:

- time-cell literature,
- sleep/dream data,
- LLM stress tests,
- sleep deprivation EEG.

Possible proxies:

- sequence coding,
- delay cost,
- anticipation,
- affect/stakes,
- interruption sensitivity.

False positives:

- timestamps equal lived duration.

## 5. Information Persistence Vs Owned Continuity

Core question:

> When does residue preserve meaning without preserving the owner?

Candidate datasets:

- cultural transmission literature,
- archive-based cases,
- AI/copy simulations.

False positives:

- information survival equals self-survival.

False negatives:

- residue without owner is meaningless.

## 6. Boundary Collapse / Ownership Alteration

Core question:

> What happens when self/world boundary, body ownership, or agency ownership weakens?

Candidate datasets:

- future body ownership illusion literature,
- depersonalization/derealization literature,
- agency illusion literature.

False positives:

- altered ownership means no consciousness.

False negatives:

- normal function means intact ownership.

## 7. DCP Transition Modeling

Core question:

> Can commitment/compression transitions be modeled without pretending they prove consciousness?

Candidate datasets:

- simulations first,
- later COGITATE/anesthesia/DREAM proxies.

Possible proxies:

- state-space narrowing,
- transition thresholds,
- branch count,
- residue,
- irreversible path selection.

False positives:

- decision compression equals lived ownership.

## 8. LIP / DCP / EIP Transition Stack

Core question:

> Can local incompleteness, decision compression, and epistemic irreversibility be separated as before/during/after components of temporal commitment?

Subquestions:

- What local evidence is insufficient before commitment?
- What pressure collapses possibilities into a committed path?
- What residue remains after commitment?
- What would count as owned crossing rather than merely state transition?

False positives:

- treating local uncertainty as proof of consciousness,
- treating decision compression as proof of ownership,
- treating residue as owner survival,
- treating DCP as universal because it resembles the operator's cognition.
