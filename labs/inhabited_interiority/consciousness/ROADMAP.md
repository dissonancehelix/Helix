## Phase 2: Rival theory comparison

Goal: Run the full case battery against all 10 rival theories. Identify overclassification
and underclassification patterns. Produce rival_theory_comparison.md report.

**Key questions:**
- Does IIT overclassify markets, games, and institutions via high phi?
- Does GNWT underclassify dreaming and minimal awareness (non-broadcast states)?
- Does higher-order theory handle blindsight better or worse?
- Does recurrent processing theory handle the ketamine/propofol contrast correctly?
- Does attention schema theory misclassify action fields as phenomenal?

## Phase 3: Threshold refinement

Goal: Address the threshold problem for D ≈ 0 (minimal awareness) and for partial-field
cases (blindsight scotoma, split-brain hemisphere). Develop a threshold model or formally
acknowledge the formula's limits.

**Options under consideration:**
- Floor threshold: C does not collapse below some minimum when D ≈ 0 but G is present
- Local vs. global field scoring: allow scotoma-level scoring separate from global C
- Null-field model: distinguish "C unresolvable" from "C = 0" more rigorously

---

## Phase 4: G-O interface formalization

Goal: Formalize the relationship between G (givenness) and O (ownership) in dissolution
and depersonalization states. Address the question: can G be present when O ≈ 0?

**Options:**
- G requires O > 0 as a structural condition → ego dissolution requires G < 1.0
- G is independent of O → ego dissolution is a genuine G-without-O state; need formal
  account of what G is given to
- G and O jointly constitute a different kind of state in dissolution → the formula
  does not apply cleanly; need a dissolution-specific model

---

## Phase 5: Empirical proxy hardening

Goal: Identify the most rigorous available proxies for each formula term. Write a formal
empirical_proxy_map document. Identify which claims are empirically testable in principle
with existing neuroscience tools and which require tools that do not yet exist.

---

## Phase 6: External review preparation

Goal: When at least one Tier 1 claim has survived the full test battery without absorbing
failures as confirmation, produce docs_for_external_review/ documents for feedback.

Not a publication target — an external-review target. Feedback from anyone who has read
both the HARDNESS_PROTOCOL.md and the full case battery.

## Long-term questions (no timeline)

- Can a version of this theory be formalized mathematically in a way that produces
  distinct, testable predictions?
- Is there an experimental protocol that could test the G-independence claim in humans
  that would be novel relative to existing no-report paradigm work?
- Can the theory be applied to non-human animal consciousness cases with more precision
  than current approaches?
- What would a direct G-test look like if one were possible?
