# Game Negative Controls

This report is a Pass 2 sidecar for `domains/self/reports/gpt_history_pass_02_world_inhabitation_negative_controls.md`.

## Core Pattern

The operator is not predicted by genre, difficulty, or complexity alone. Game fit depends on world-inhabitation: readable field pressure, safe/inviting entry, role/mission structure, re-entry, durable world memory, and freedom from untrusted social volatility.

## Negative Controls

| Game / genre | Violated gate | Interpretation |
|---|---|---|
| Counter-Strike | exposure/anxiety gate | Simple bomb structure is not enough; peeking, snipers, one-shot exposure, and personal punishment lower invitation. |
| Battle royale | durable-world gate | The format has low pull; Fortnite is funny mainly as absurd cameo/cosmetic collision. |
| Fighting games | duel exposure gate | Too personal, too 1v1, too mechanical; opponent attention is concentrated directly on the operator. |
| Card games | abstract duel gate | Mechanical/personal duel pattern with low inhabitable field pull. |
| Rust / TTT / DarkRP | trusted-world gate | Untrusted human volatility turns the field into an anxiety factory. |
| Most survival/crafting | motivation gate | Crafting/building alone is not enough; survival pressure can become chores or anxiety. |
| Superhero movies / MCU | continuity gate | Repetitive plot texture and brand aggregation can mimic continuity without world-memory. |

## Positive Anchors

- Ace Combat: structured military mission field, atmosphere, role identity.
- SOCOM: tactical/role military identity without random social-volatility emphasis.
- Terraria: exploration and depth override survival/crafting aversion; base-building can become durable inhabitation.
- Trails: narrative-first continuity and world-memory.
- Yume Nikki / PilotRedSun: private atmospheric re-entry and interior texture.
