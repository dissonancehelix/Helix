# Inhabited Interiority Research Program

Working title: **Inhabited Interiority**  
Status: Active, pre-publication  
Version: 0.3  
Last revised: 2026-04-26

---

## Central question

What makes a structured field become inhabited from within, and what must be
preserved for that inhabited interior to remain the same self through time?

---

## Core thesis (under test)

**Consciousness is inhabited interiority.**  
**Self is inhabited interiority carrying ownership through time.**

These are falsifiable claims. This program is designed to stress-test them.

---

## Core formulas

### Consciousness

    C = G · U · D · T · F

| Term | Name | Description |
|------|------|-------------|
| G | Givenness | Appearance-from-within. Primitive residue. Not directly detectable. |
| U | Unity | Local co-presence of field contents |
| D | Differentiation | Qualitative richness and complexity |
| T | Temporal thickness | Specious present; continuity across time |
| F | Field inclusion | States are phenomenally included in the field |

Behavior: if any term = 0, C = 0. If any term = null, C = unresolvable.

### Self-continuity (active)

    S = C · P · O · B · K · N

| Term | Name | Description |
|------|------|-------------|
| C | Consciousness | From formula above |
| P | Perspective | From-here; spatiotemporal anchor |
| O | Ownership | Mineness; states felt as one's own |
| B | Self-boundary | Inside/outside distinction |
| K | Carried constraint | Past active as constraint on future transformation |
| N | Non-branching | Singular causal continuity; branching is fatal |

Hard stop: N = 0 overrides all other scores. S = 0 when N = 0.

### Self-continuity (suspended state)

    S_suspended = P · O · B · K · N · R

Used when C is interrupted (sleep, anesthesia). R = restart continuity (same causal
process resumes the field). C is set aside during the gap; R captures whether the
gap is bridged by the same causal chain.

## Research questions

**Tier 1 — Central**

1. Can C = G·U·D·T·F correctly classify the standard edge cases (waking, dreaming,
   anesthesia, locked-in syndrome, minimal awareness, psychedelic dissolution)?

2. Can S = C·P·O·B·K·N correctly handle branching, gradual replacement, suspension,
   and ownership loss?

3. Does the false-positive firewall correctly reject games, institutions, LLMs,
   fictional characters, and collective systems?

4. Which rival theories overclassify or underclassify more severely than inhabited
   interiority theory?

**Tier 2 — Structural**

5. What is the relationship between G and O in ego dissolution and depersonalization?
   (Can G be present when O ≈ 0?)

6. What threshold problem exists when D ≈ 0 in minimal awareness? Does the
   multiplicative formula break down at the low end?

7. What precisely distinguishes "agent resuming from memory" from "self resuming
   from sleep"? (The suspension formula asserts a difference; the mechanism is
   underspecified.)

8. In non-destructive copying (N = 0 for both), does the original's stronger causal
   claim create a meaningful asymmetry the formula obscures?

**Tier 3 — Speculative**

9. Is G detectable at all, even indirectly? What would a positive G-proxy look like?

10. Can the theory be refined to handle partial-field cases (blindsight scotoma,
    split-brain hemisphere, absence-seizure gaps) with more precision than "G locally
    absent"?

11. Does ego dissolution prove that G does not require O, or does it prove that what
    seems like G without O is a different kind of state entirely?

## Program structure

The research program has three modes:

**Classification mode**: Run cases through the formula and classification system.
Output: case verdict + confidence + preserved breaks.

**Falsification mode**: Apply the six tests to candidate claims. Output: claim status
(candidate / weakened / broken / unresolved).

**Comparison mode**: Evaluate rival theories against the same case battery. Output:
overclassification count, underclassification count, empirical proxy quality,
falsifier clarity.

## Known unresolved problems

1. **G-O interface in dissolution states**: If G = appearance-from-within, and dissolution
   removes the "within," what is G given to? Unresolved.

2. **Threshold underdetermination**: The multiplicative formula gives C ≈ 0 when D ≈ 0.
   Minimal awareness cases report something when D is near zero. Unresolved.

3. **Non-destructive copy asymmetry**: N = 0 assigns same verdict to original and copy.
   Original may have stronger causal claim. Formula does not capture this. Unresolved.

4. **Suspension mechanism**: Theory claims sleep/anesthesia preserves self-continuity via
   causal process continuity. The precise mechanism distinguishing this from agent memory
   resumption is underspecified. Unresolved.

5. **Split-brain U score ambiguity**: U can be scored from neural architecture (low) or
   from verbal report (high). The theory has not resolved which criterion U tracks.
   Unresolved.

6. **G-detection gap**: The theory cannot specify what would constitute positive evidence
   for G in any novel system. Without this, all verdicts of G > 0 are inferences from
   convergent proxies rather than derived conclusions. Acknowledged, not resolved.

## Empirical anchors

- **No-report paradigm** (Tsuchiya et al. 2015): Behavioral report can contaminate
  neural correlates of consciousness. Consciousness must be separable from report.

- **PCI / TMS-EEG** (Casali et al. 2013): Perturbational complexity index as behavioral-
  report-independent proxy for C. Near-zero in propofol; elevated in ketamine and dreaming.

- **COGITATE adversarial collaboration** (Cogitate Consortium 2023): Preregistered
  predictions, explicit pass/fail conditions. Model for this research program.

- **Owen et al. 2006** (imagining tennis): Covert consciousness detection in disorders
  of consciousness. Empirical anchor for G-independence from behavioral output.
