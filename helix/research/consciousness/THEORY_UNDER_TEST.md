## Core Research Question

> What makes a structured field become **inhabited from within**, and what must be preserved for that inhabited interior to remain the same self through time?

This is not the same as asking what makes a system intelligent, complex, self-organizing, or persistent. Many systems have those properties without any established evidence of appearance-from-within.

---

## The Hard Stop

**G / givenness may be the primitive residue.**

Givenness (previously labeled A in v0.x fixtures) — appearance-from-within, the "what it is like" — is where conceptual analysis stops and empirical testing begins. The harness cannot detect G directly. No behavioral test, no functional criterion, no information-theoretic measure currently established can read off whether givenness is present.

What the harness *can* do:

1. Classify cases by the structural features that are *necessary* for G (according to the theory).
2. Test whether systems without G candidates are incorrectly promoted by the theory (false positive scan).
3. Identify cases where other components (U, D, T, F) are present while G is contested.
4. Preserve the gap between *classifiable features* and *G itself*.

This is the epistemological architecture of the lab. The harness is not a G-detector. It is a classification and falsification system that operates around G without pretending to measure it.

## Formulas (Stress-Test Scaffolds, Not Proofs)

### Phenomenal Consciousness

```
C = G · U · D · T · F
```

| Term | Meaning |
|---|---|
| **G** | Givenness / appearance-from-within. The primitive. Cannot be detected behaviorally. |
| **U** | Local unity / co-presence. Internal states co-present in a unified field. |
| **D** | Differentiation / internal contrast. Field contains distinguishable qualitative states. |
| **T** | Temporal thickness. Specious present — a now with retention and protention. |
| **F** | Field inclusion. States included in active phenomenal field (not merely processed). |

If any term = 0, C = 0 (collapses).  
If any term = null, C = unresolvable.

**V0.x note:** G was labeled A (appearance-from-within) in earlier fixtures. Same component, renamed to emphasize its primitive status.

### Self-Continuity

```
S = C · P · O · B · K · N
```

| Term | Meaning |
|---|---|
| **C** | Consciousness / phenomenal fielding (from C above) |
| **P** | Perspective / from-here |
| **O** | Ownership / mineness |
| **B** | Self-boundary |
| **K** | Carried constraint through time |
| **N** | Non-branching continuity |

If N = 0, singular self-continuity is impossible regardless of other scores.  
If C = 0, no self is possible.

### Suspended-State Self-Continuity

```
S_suspended = P · O · B · K · N · R
```

Where **R** = restart continuity — whether the phenomenal field resumes via the **same causal process** across a gap (sleep, anesthesia). C is set aside for the duration of the gap; R carries the continuity claim.

### Personhood

```
Person = S · M · Rv · V · Gv
```

| Term | Meaning |
|---|---|
| **M** | Autobiographical memory |
| **Rv** | Reasons-responsive agency |
| **V** | Values / future self-concern |
| **Gv** | Social recognition / governance attribution |

### Transfer Test

```
Transfer(X_t → Y_t+1) preserves S iff it preserves C, P, O, B, K, and N.

Copy(pattern) ≠ Preserve(Self)
```

A portable self-continuity carrier must preserve **inhabited ownership**, not merely information, memory, or symbolic pattern.

## Six Falsification Tests

1. **Temporal deletion** — Remove T. What remains? Self and personhood should collapse.
2. **Boundary perturbation** — Disturb B and O. What separates C from S?
3. **Network partition** — Split the system. Two selves or a partition?
4. **False-positive field** — What has structure without G? Does the theory correctly reject it?
5. **Carried constraint** — Does K distinguish identity from mere pattern?
6. **Geometry/topology** — What ownership relations survive deformation?

## Key Language to Preserve

- Consciousness is inhabited interiority.
- Self is inhabited interiority carrying ownership through time.
- Field-like structure is not automatically phenomenal.
- Pattern survival is not field survival.
- Field survival is not automatically self survival.
- Branching is fatal to singular self-continuity.
- A portable self-continuity carrier must preserve inhabited ownership, not merely information, memory, or symbolic pattern.
- Givenness / appearance-from-within may be the primitive residue where conceptual analysis stops and empirical testing begins.

## Source Anchors

- **No-report paradigms** — Behavioral report can contaminate and confound neural correlates of consciousness. The no-report paradigm (Tsuchiya et al. 2015) attempts to separate these.
- **PCI/TMS-EEG** — Perturbational complexity index (Casali et al. 2013) can help discriminate minimally conscious from vegetative/unresponsive states without relying on behavioral report.
- **COGITATE adversarial collaboration** — Theory-neutral adversarial tests between IIT and GNWT (Melloni et al. 2023). Result: partial support for GNW's reportability claim; IIT's Φ predictions not supported in tested regions. Illustrates how theories need falsifiable predictions.
- **AI consciousness indicators** — Current AI systems are not established as conscious. Future evaluation requires theory-derived indicators beyond self-report, behavioral coherence, or claimed inner states.
- **SEP: Phenomenological Self-Consciousness** — prereflective self-consciousness / for-me-ness. https://plato.stanford.edu/entries/self-consciousness-phenomenological/
- **SEP: Temporal Consciousness** — specious present, retention/protention. https://plato.stanford.edu/entries/consciousness-temporal/
- **SEP: Personal Identity** — psychological continuity, fission, non-branching. https://plato.stanford.edu/entries/identity-personal/
- **Josipovic & Miskovic 2020** — Nondual awareness and minimal phenomenal experience.
- **Gamma & Metzinger 2021** — MPE questionnaire.
- **Perich, Narain, Gallego 2025** — Neural manifold view of the brain.
