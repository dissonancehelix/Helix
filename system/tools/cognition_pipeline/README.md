# cognition_pipeline

Role: `stage_helper`.

Small cognitive-mapping experiment pipeline used by labs. Start from `system/tools/TOOL_INDEX.yaml`; use `python -m system.tools.cognition_pipeline.e2e` for the end-to-end smoke path.

Outputs belong under `reports/cognition/` or a lab-specific report path. Do not promote results into `model/` without review.
