# nsf2vgm
This batch NSF to VGM converter, based on the modified NEZPlug++ VGM mod, can automatically split tracks according to the m3u playlist, and can directly read 7z packages downloaded from nsf.joshw.info and automatically convert them to VGM before packaging them into a zip file. Currently supported chips are 2A03 (including DMC), FME7/S5B (via YM2149), and VRC7 (via YM2413).
