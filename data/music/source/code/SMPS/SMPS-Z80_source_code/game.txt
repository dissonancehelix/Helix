	１：ＪＯＹポートアクセス時（ＳＷＩＴＣＨ＿ＲＥＡＤ）の注意事項
	２：Ｚ８０メモリアクセスルーチンの注意事項
	３：Ｚ８０の　リスタートについて
	

	１：ＪＯＹポートアクセス時（ＳＷＩＴＣＨ＿ＲＥＡＤ）の注意事項

		　追補マニュアルで記されているように、＄０Ａ１００ＸＸアクセスの際には、Ｚ８０へＢＵＳリクエストをかける必要があります。したがって、ＪＯＹポートアクセス時には必ず、Ｚ８０へＢＵＳリクエストをかけて下さい。

	２：Ｚ８０メモリアクセスルーチンの注意事項

		　Ｚ８０側へのメモリアクセス、特にサウンドリクエストコード書き込みをＭＡＩＮルーチンで行う場合、インタラプトディスエブル状態にしてください。これは、Ｚ８０へのバスリクエストを要求し、データを書きにゆく間にインタラプトがかかり、ＳＷＩＣＨ　ＲＥＡＤ後のＢＵＳリクエスト解除が行われることがないようにするためです。
		
		
		

	３：Ｚ８０の　リスタートについて
		　Ｚ８０にＲＥＳＥＴ要求をし、それを解除する間、ＮＯＰ１４個分のＷＡＩＴが必要です。
